# 🏥 DiagnoAI — Multi Disease Prediction System

A machine learning web app that predicts 
3 diseases using patient data.

## 🎯 Diseases Covered
- 🎗️ Breast Cancer
- 🩺 Diabetes  
- ❤️ Heart Disease

## 🛠️ Tech Stack
- Python
- Scikit-learn
- XGBoost
- Streamlit
- Pandas & NumPy

## 📊 Model Performance
| Disease | Model | Accuracy |
|---------|-------|----------|
| Breast Cancer | Logistic Regression | 97.37% |
| Diabetes | Random Forest | 76.62% |
| Heart Disease | Random Forest | 86.96% |
